1.CRLF

  • 1.CRеӣһв»Ӣз¬ҰеҸ·Carriage ReturnпјҢ е°Ҷе…үж Үз§»еҠЁеҲ°вҫҸвҫёпјҢ дҪҶдёҚжҚўвҫҸпјҢиҪ¬д№үз¬ҰеҸ·\r,еҚҒиҝӣеҲ¶ж•°13жҲ–еҚҒе…ӯиҝӣеҲ¶0x0d
  • 2.LFжҚўвҫҸз¬ҰLine Feed(newline)пјҢ е°Ҷе…үж Үз§»еҠЁеҲ°дёӢвјҖвҫҸпјҢиҪ¬д№үз¬ҰеҸ·\n, еҚҒиҝӣеҲ¶ж•°10,еҚҒе…ӯиҝӣеҲ¶ж•°0x0a
  • 3.WindowsдҪҝвҪӨCRLFиЎЁвҪ°дёӢвјҖвҫҸпјҢ UNIXе’ҢLINUXдҪҝвҪӨLF иЎЁвҪ°дёӢвјҖвҫҸпјҢ MACOSдҪҝвҪӨCRиЎЁвҪ°дёӢвјҖвҫҸ

2.GITеҜ№жҚўиЎҢз¬Ұзҡ„еӨ„зҗҶ

жҲ‘д»¬д»ҺеүҚвҫҜеҸҜд»ҘзҹҘйҒ“CRLFе’ҢLFжҳҜдёҚеҗҢзҡ„еӯ—з¬ҰиЎЁвҪ°пјҢ еӣ жӯӨзјәе°‘йҖҡвҪӨвҫҸз»“е°ҫдёә git зӯү зүҲжң¬жҺ§еҲ¶иҪҜд»¶еёҰжқҘдәҶвҝҮзғҰпјҢ иҝҷзұ»иҪҜд»¶дҫқиө–дәҺвҫ®еёёзІҫзЎ®зҡ„еӯ—з¬ҰвҪҗиҫғжқҘзЎ®е®ҡвҪӮд»¶вҫғдёҠж¬Ў зӯҫвјҠд»ҘжқҘжҳҜеҗҰеҸ‘вҪЈдәҶеҸҳеҢ–гҖӮ еҰӮжһңејҖеҸ‘вјҲе‘ҳдҪҝвҪӨ Windows е№¶дё”еҸҰвјҖдёӘдҪҝвҪӨ Mac жҲ– LinuxпјҢ д»–д»¬жҜҸдёӘвјҲйғҪдҝқеӯҳе’ҢжҸҗдәӨзӣёеҗҢзҡ„вҪӮд»¶пјҢ д»–д»¬еҸҜиғҪдјҡеңЁд»–д»¬зҡ„ git diff дёӯзңӢеҲ° вҫҸз»“жқҹжӣҙж”№вҖ”вҖ”д»Һ CRLF еҲ° LF зҡ„иҪ¬жҚў

еңЁgitдёҠжҸҗдәӨж—¶жҳҜеҗҰиҝӣвҫҸвҫҸе°ҫиҪ¬жҚўautocrlfдёәtrueж—¶дјҡеңЁwindowsдёӢpushж—¶е°ҶCRLFвҫғеҠЁжӣҝжҚўдёәLF,еңЁжӢүеҸ–ж—¶е°ҶLFвҫғеҠЁжӣҝжҚўдёәCRLF,inputж—¶жҸҗдәӨж—¶иҪ¬жҚўдёәLFдҪҶжӢүеҸ–ж—¶дёҚеҒҡд»»дҪ•иҪ¬жҚўгҖӮ
