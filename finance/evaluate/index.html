# ATR指标

ATR（Average True Range，平均真实波幅）是由J. Welles Wilder Jr.于1978年开发的技术分析指标，用于衡量价格的波动程度。

# 计算方法

  1. 计算真实波幅（TR）：取以下三者中的最大值：

    • 当前高点与前一日低点的差值
    • 当前高点与前一日收盘价的绝对值
    • 当前低点与前一日收盘价的绝对值

  2. 计算ATR：通常使用14日简单移动平均

ATR=1n∑i=1nTRi

# 应用场景

  • 衡量波动率：ATR值越大，表示市场波动越剧烈
  • 设置止损：可根据ATR倍数（如2倍ATR）动态调整止损位
  • 仓位管理：波动率越大，仓位应越小以控制风险
