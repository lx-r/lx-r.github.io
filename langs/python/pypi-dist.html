python дёӯзҡ„еҢ…еҲҶжҲҗдёӨз§ҚпјҢдёҖз§ҚжҳҜжҷ®йҖҡеҢ…regular packageпјҢдёҖз§ҚжҳҜе‘ҪеҗҚз©әй—ҙеҢ…namespace packageгҖӮ

# 1.1 Regular Package

жҷ®йҖҡеҢ…жҳҜжҢҮеҢ…еҗ«дёҖдёӘ __init__.pyж–Үд»¶зҡ„жҷ®йҖҡзӣ®еҪ•,еңЁPython 3.2еҸҠд№ӢеүҚзҡ„зүҲжң¬дёӯдёҖзӣҙдҪҝз”Ёзҡ„йғҪжҳҜиҝҷз§ҚеҢ…гҖӮжҷ®йҖҡеҢ…еңЁеҜје…Ҙзҡ„ж—¶еҖҷпјҢдјҡйҡҗејҸиҮӘеҠЁи°ғз”ЁеҢ…дёӯзҡ„__init__.pyж–Үд»¶пјҢ__init__.pyдёӯе®ҡд№үзҡ„еҜ№иұЎпјҢеҸҳйҮҸйғҪдјҡиў«йҷҗеҲ¶еңЁеҢ…еҗҚзҡ„е‘ҪеҗҚз©әй—ҙдёӯгҖӮ

еҰӮдёӢеҪўејҸзҡ„еҢ…пјҡ

жҖ»з»“пјҢеҸӘйңҖиҰҒи®°дҪҸпјҢ__init__.pyжҳҜеҢ…еҜје…Ҙж—¶жңҖе…Ҳжү§иЎҢзҡ„ж–Үд»¶гҖӮ
