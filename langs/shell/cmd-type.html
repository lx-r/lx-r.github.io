# shell的内建命令与外部命令

# 外部命令

外部命令，有时候也被称为文件系统命令，是存在于bash shell之外的程序。它们并不是shell
程序的一部分。外部命令程序通常位于/bin、/usr/bin、/sbin或/usr/sbin中。

(base) xx@xx-rob:~$ type head 
# head is hashed (/usr/bin/head)
(base) xx@xx-rob:~$ type ps
# ps is hashed (/usr/bin/ps)

外部命令执行时，会创建出一个子进程。这种操作被称为衍生（forking）。

# 内建命令

内建命令不需要使用子进程来执行。它们已经和shell编译成了一体，作为shell工具的组成部分存在。不需要借助外部程序文件来运行。

内建命令的执行速度要更
快，效率也更高。

查看一个命令是内建命令还是外部命令，使用type:

type cd

# cd is a shell builtin

一个有用的内建命令是 history 命令,用来查看最近用过的命令列表：

  • !!可唤回刚使用过的命令
  • 修改HISTSIZE可以设置history的条数，默认是1000条
  • history列表的命令存在于.bash_history文件中

可以在退出shell会话之前强制将命令历史记录写入.bash_history文件。要实现强制写入，需要使用 history 命令的 -a选 项。
