# 傅里叶变换

# 存在条件

傅里叶变换存在的条件要求原函数绝对可积：

∫−∞∞|f(t)|dt<∞

也就是说：

limt→±∞f(t)=0

其实也很容易理解，

考虑最简单的一元函数，是函数与轴之间围起来的面积，如果要满足，则时必须无限趋近于轴，也即。

# 傅里叶变换与反变换

的傅里叶变换，从时域到频率域：

F(f)=f^(ω)=∫−∞∞ejωtf(t)dt

的反傅里叶变换，从频率域到时域：

F−1(f^(ω)=∫−∞∞e−jωtf^(ω)dω

# 导数的性质

# 导数的傅里叶变换

F(f′(t))=∫−∞∞ejωtf′(t)dt=ejωtf(t)|−∞∞+jω∫−∞∞ejωtf(t)dt=jω∫−∞∞ejωtf(t)dt=jωF(f)

值得注意的是在公式中的第一项，根据傅里叶变换的存在条件，因此其结果为0，因此可以推导得到上面的公式。
