# 1.下标

# 1.1 问题

在使用LaTex进行排版时，一个常见的需求是要把下标放在某个文字或者符号的正下方：

∑i=1Nxi

正常情况下可使用：

\sum\limits_{i=1}^{N}x_i

但是\limits命令要求\sum部分必须的数学符号，否则会报错：

! Limit controls must follow a math operator.

# 1.2解决

解决方法是用\mathop{expres}命令将expres转化成数学符号，写成

\mathop{max}\limits_{i}{y_i}

结果：

maxi⁡yi
